## Running instructions

### Make sure you have:
- Node.js 18+
- Rust 1.75+
- A [Codeforces](https://codeforces.com) account.

### Development
To get the front end running 

1. **Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
To get the backend working

2. **Backend:**
   ```bash
   cd backend
   cargo run
   ```

3. Open `http://localhost:3000` (mostly)

Take a look at 
- [**Game Rules**](rules.md): For the rules
